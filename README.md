MIDI Chord Explorer lets you play notes from a MIDI keyboard (or your computer keys) and instantly see what chord you‚Äôve created. It suggests missing notes, voice-leading steps, scale context, and musical directions you can explore.

Key Features

- Chord Detection: Press notes and the app identifies possible chords.
- Partial Chords Explained: If you are missing tones, it shows what to add.
- Where Next: Suggests musical next chords using actual harmonic functions.
- Voice Leading: Shows how to move between chords smoothly (keep / drop / add).
- Scale Context: Shows which scales fit the chord and lists their notes.
- Bookmarking: Save chords you like using the ‚≠ê star icon.
- Copy to Clipboard: Copy the full chord card text, including voice-leading.
- MIDI Input: Works with USB MIDI synths.
- Keyboard Input: Use your computer keys if you don't have MIDI input.

Browser Support
